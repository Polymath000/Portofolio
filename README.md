## GitHub Pages

If you host this Flutter web app on GitHub Pages, make sure the site is built
with the repository path as the base href.

- The included workflow builds with `--base-href "/<repo-name>/"` and deploys
  the contents of `build/web`.
- If you deploy manually, upload `build/web`, not the source `web/` folder.
- In GitHub Pages settings, choose `GitHub Actions` as the source.

The 404s for `manifest.json` and `flutter_bootstrap.js` usually mean the app
was built for `/` instead of the GitHub Pages subpath.
